# DiscMathlib

DiscMathlib is a lightweight library for solving several discrete mathematics tasks.

## Features

The library provides:

1. Representation of sqrt(N) as a periodic continued fraction
2. Computation in modular arithmetic (Zn)
3. Root finding using the extended Euclidean algorithm
